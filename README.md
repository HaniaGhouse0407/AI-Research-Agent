## 🧠 Overview

A five-agent LangGraph pipeline that autonomously searches arXiv and Semantic Scholar, reads papers, synthesises findings, critiques methodology, and writes a structured research report — all from a single natural language query.

---

## ✨ Features

- ✅ 5 specialised agents: Retrieval, Reading, Synthesis, Critique, Report
- ✅ arXiv API + Semantic Scholar integration
- ✅ LangGraph state machine with conditional routing
- ✅ Real-time agent activity visualisation
- ✅ Structured Markdown report output
- ✅ Configurable search depth and paper count

## 🚀 Quick Start

```bash
# 1. Clone
git clone https://github.com/HaniaGhouse0407/AI-Research-Agent.git
cd AI-Research-Agent

# 2. Install dependencies
pip install -r requirements.txt

# 3. Set environment variables (if needed)
cp .env.example .env
# Edit .env with your API keys

# 4. Run
streamlit run app.py
```
